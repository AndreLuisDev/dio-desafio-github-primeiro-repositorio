# PHP

https://www.youtube.com/watch?v=UCIZfVKmXNU

	- linguagem de programação considerada um Script
		- LP interpretada e não-compilada
		- pega-se o código-fonte PHP
			- interpretada no servidor web (linguagem programação web)
			- traduzida para Binário
			- processada pelo Brownser em HTML
				- forma rápida
			- PHP x Astah
			- Versão 4.0 em diante - orientação a objetos
			- O PHP "roda por debaixo" do Servidor Web Apache
				- interpretando todas as estruturas de comando
				- lendo o Bando de Dados MySQL
				- e resolvendo em HTML no Browser de quem faz a requisição do site
			- cria conteúdo dinâmico

			- Ambiente de desenvolvimento LAMP
				CSS, HTML, JavaScript, MySQL, Apache
