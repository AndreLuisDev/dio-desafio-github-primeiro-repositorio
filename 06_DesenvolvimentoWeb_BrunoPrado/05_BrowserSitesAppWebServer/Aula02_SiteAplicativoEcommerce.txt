# APLICATIVO
	- Software executado no navegador
	- App é uma espécie de um navegador
	- Youtube 
		No início era só um site
		Hoje é um App no celular
			- como se fosse um link do site Youtube

	- Facebook do celular
		App é um link do Facebook

	- APP == SITES
		Aplicativos são pequenos softwares que rodam no navegador
			- editor video/imagem, bate papo - ONLINE

		- Hoje em dia, vê-se menos sites rudimentares
			e vê-se mais sites com função de App

	- PROGRAMA/SOFTWARE/SITE == APLICATIVO (conotação)
		- sites são app's

# E-COMMERCE

	- comércio eletrônico
		sites de compra e venda com sistema de pagamento

	- o sistema de pagamento pode ser externo ao site
		- PicPay, PagSeguro, PayPal, Mercado Pago

	 - não é um sistema nativo do próprio site que conduz o sistema de pagamento
		- fica a cargo dos sistemas externos

- infográfico de como são feitas as transações de pagamento em outros países

- Boleto só há no Brasil
- Débito/Crédito existem em diversos países
- Hoje em dia o sistema de pagamento dos sites
	são sistemas externos de pagamento conforme mencionado anteriormente
